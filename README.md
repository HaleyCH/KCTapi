# KCT-api Document

## What is KCT-api
`KCT-api`(KFC Crazy Thursday api) is an easy-install api
using `FastAPI`, providing stable services related to KCT.

## Installation
1. `git clone https://github.com/HaleyCH/KCTapi`
2. `pip install -r requiremenst.txt`
3. `python main.py`
